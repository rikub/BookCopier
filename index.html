What is it?

Book copier is a Minecraft plugin for Bukkit API. It's completely open-source. You can browse, download and integrate my source to your project.

Features

With BookCopier is administration of writable books much easier! You can copy or send books to targer user.

Commands

<> - required arguments [] - optional arguments

/bc help - Shows commands and info about plugin.
/bc copy [value] - Copies book in your hand.
/bc give <player> [value] - Gives book in your hand target player.

Upcoming features

"First login book" - When player joins server for first time, plugin gives him book what you specified in file automaticaly!

You'll be able to load books from file!
