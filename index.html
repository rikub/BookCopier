What is it?

Book copier is a Minecraft plugin for Bukkit API. It's completely open-source. You can browse, download and integrate my code to your project.

Features

With BookCopier is administration of writable books much easier! You can copy or send books to target user and backup them to files. You can naturally load them back.

How to save book to file

It's simpe. Only type /bc save when you're holding writable book. File with backup will be saved to plugins/BookCopier/fileName.book

How to load book from file

Type /bc load . File named "fileName" must be in directory plugins/BookCopier.

How to edit or create .book files

.book format is standart .txt only different named. You can open it in Notepad or Notepad++. If you wont to create it you must abide few rules. This is conception of file:

title: default
author: default
pages: page1,page2

Pages are separated by comma. Maximum are 20 pages (limited by MC).

Commands

<> - required arguments [] - optional arguments

/bc help - Shows commands and info about plugin.
/bc copy [value] - Copies book in your hand.
/bc give <player> [value] - Gives book in your hand target player.
/bc load <fileName> - Loads book from file to your inventory.
/bc save <fileName> - Saves book in your hand to file.

Upcoming features
